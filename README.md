# All Time Performance (ATP)

All Time Performance (repo named time-bandits) is a community-driven platform for athletes to showcase their verified attempts on rugged routes.

## Features

Currently functioning as a **frontend prototype**, with plans to evolve into a full-fledged application including:

- **Verified Attempts**: Every performance is authenticated for credibility and fairness.
- **Standardized Routes**: Curated definitions ensure consistency across athletes.
- **Transparent Scoring**: Dynamic rankings evolve with participation and performance density.
- **Discoverability**: Explore routes, leaderboards, and athlete rankings with ease.

- [React](https://react.dev/)
